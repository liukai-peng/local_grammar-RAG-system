# 局部语法RAG桌面助手使用指南

## 🚀 快速开始

### 1. 启动应用
```bash
python rag_desktop_app.py
```

### 2. 首次配置
1. 打开"系统设置"标签页
2. 输入你的Deepseek API密钥（在https://platform.deepseek.com/ 申请）
3. 确认其他路径配置：
   - 向量库路径：`./chroma_db_merged`（默认已配置）
   - 本地模型路径：`./bge-large-zh-v1.5`（默认已配置）
4. 点击"保存设置"
5. 点击"重启RAG系统"

### 3. 开始查询
1. 回到"智能查询"标签页
2. 在输入框中输入问题，例如："什么是局部语法？"
3. 点击"查询"按钮或按回车键
4. 等待几秒后即可看到答案和相关参考文献

## ✨ 功能特色

### 🤖 智能查询
- 支持自然语言提问
- 基于7万+篇局部语法文献回答
- 答案附带参考文献来源和页码
- 显示文献相似度

### ⚙️ 灵活配置
- 可自定义检索结果数量（1-20条）
- 可调整生成温度（0-1，越低越稳定）
- 支持配置不同的LLM模型
- 可切换不同的向量数据库

### 📜 历史记录
- 自动保存所有查询历史
- 双击历史记录可重新查看结果
- 支持清空历史记录

## 📁 文件说明

| 文件 | 功能 |
|------|------|
| `rag_desktop_app.py` | 主程序文件 |
| `config.json` | 配置文件（自动生成） |
| `requirements_app.txt` | 依赖清单 |
| `chroma_db_merged/` | 合并后的向量数据库 |
| `bge-large-zh-v1.5/` | 本地嵌入模型 |
| `static/` | PDF文献库 |

## 🛠️ 打包成EXE（可选）

如果你想打包成独立的exe文件，不需要Python环境也能运行：

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "局部语法RAG助手" --add-data "chroma_db_merged;chroma_db_merged" --add-data "bge-large-zh-v1.5;bge-large-zh-v1.5" --add-data "static;static" rag_desktop_app.py
```

打包后的文件在 `dist/` 目录下。

## 📝 注意事项

1. **API密钥**：首次使用必须配置Deepseek API密钥
2. **网络要求**：生成答案需要网络连接（检索功能可以离线使用）
3. **查询速度**：首次查询可能较慢，后续查询会变快
4. **存储空间**：整个程序约1.5GB（包含模型和数据库）

## ❓ 常见问题

**Q: 提示"RAG系统初始化失败"怎么办？**
A: 检查向量库路径和模型路径是否正确，确保文件夹存在。

**Q: 查询速度慢怎么办？**
A: 可以适当减少检索结果数量，或者升级电脑配置。

**Q: 答案不准确怎么办？**
A: 可以尝试降低生成温度（比如设置为0.1），或者增加检索结果数量。

**Q: 可以添加新的文献吗？**
A: 目前版本需要使用原来的RAG管道重新构建向量库，后续版本会添加图形化的文献导入功能。
